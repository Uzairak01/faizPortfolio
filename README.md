# Faiz Portfolio

A professional, fully responsive front-end portfolio website created for Faiz Aslam Khan. This project serves as my very first front-end web development project, showcasing modern web design principles, structured layout, and interactive elements.

## 🚀 Features

- **Responsive Design:** Fully optimized for desktops, tablets, and mobile devices.
- **Modern UI/UX:** Features a sleek slate and teal color palette, a glassmorphism header, and soft, elegant drop-shadows.
- **Dark/Light Theme Toggle:** Includes a fully functional theme switcher that saves user preferences using local storage.
- **Interactive Animations:** Smooth hover animations on navigation links, buttons, and profile images.
- **Organized Sections:** 
  - *Home:* Hero section with an introduction and a downloadable resume.
  - *About:* Detailed career timeline and professional summary.
  - *Services:* Key domain expertise and services offered.
  - *Contact:* A beautifully designed modern contact form.

## 🛠️ Technologies Used

- **HTML5:** Clean, semantic document structure.
- **CSS3:** Custom styling using modern layouts (Flexbox), CSS variables for theming, and transition effects.
- **JavaScript (Vanilla):** DOM manipulation tailored for the dark mode toggle and the mobile hamburger menu.

## 💻 How to Run Locally

Since this is a static website built with pure HTML, CSS, and JS, no complex build steps or dependencies are required.

1. **Clone the repository** (if using Git) or simply download the project folder.
2. **Open the project folder.**
3. **Run the site** using any local development server:
   - *Using VS Code:* Install the "Live Server" extension, right-click `index.html`, and select "Open with Live Server".
   - *Using Python:* Open a terminal in the directory and run `python -m http.server 8080`. Then visit `http://localhost:8080`.
   - *Directly:* You can also simply double-click `index.html` to open it in your default web browser.
